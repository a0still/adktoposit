# Important Links and Resources

## What systems do I need to access for shrink research?

Use these systems for shrink investigation and operational tasks:

### WAVE TASK System
**Purpose:** Walmart Automated Verification Environment - Task management and workflow system

**URL:** https://wave.prod.walmart.com/dashboard/default

**What you can do:**
- Submit miscellaneous MUMD (Markup/Markdown) forms
- Complete operational tasks and workflows
- Access process documentation

### AP1-Stores System
**Purpose:** Asset Protection reporting and analysis tools

**URL:** https://teams.wal-mart.com/sites/APReportingSharePoint/SitePages/Home.aspx

**What you can do:**
- Review Asset Protection reports
- Investigate theft indicators
- Access exception reports
- View AP queries and dashboards
